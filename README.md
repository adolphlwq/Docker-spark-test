##Spark镜像说明文档

###环境
* ubuntu14.04 64bit

###Spark镜像
#####软件环境
* 安装`openjdk`并配置环境变量
* 安装`spark-1.6.0-bin-hadoop2.6+`，已经编译好可以直接使用

#####功能
* 学习和实践Spark
* 大数据的分析和挖掘
* 部署Spark集群
* 结合其它的大数据工具使用

#####不足
* 镜像体积为1.017G，有点大，不更新基础ubuntu镜像的话大小会在920M左右，更轻量

[本镜像的github地址](https://github.com/adolphlwq/Docker-spark-test)
